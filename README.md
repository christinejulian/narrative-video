# Film Sound

A collection of classic film projector sound effects, including mechanical whirring, gear clicks, and celluloid tracking sounds.

## Overview

This repository provides clean, high-quality audio files capturing the nostalgic, mechanical ambiance of vintage film projection equipment. It is ideal for sound design, multimedia projects, historical documentaries, or creating an authentic cinema atmosphere.
